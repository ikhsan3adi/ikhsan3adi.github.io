Developing

Once you've created a project and installed dependencies with 

npm install
(or 
pnpm install
or 
yarn
), start a development server:

npm run dev

&amp;lt;span class=&lt;span class=<span class="hljs-string">&quot;hljs-string&quot;</span>&gt;&amp;quot;hljs-comment&amp;quot;&lt;/span&gt;&amp;gt;&lt;span class=<span class="hljs-string">&quot;hljs-comment&quot;</span>&gt;<span class="hljs-comment"># or start the server and open the app in a new browser tab&amp;lt;/span&amp;gt;&lt;/span&gt;</span>
npm run dev -- --open

Building

To create a production version of your app:

npm run build

You can preview the production build with 

npm run preview
.

To deploy your app, you may need to install an adapter for your target environment.
